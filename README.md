## Project Title :

Customer Relationship Management (CRM) System

## Project Description :

This CRM project is developed as part of a college academic project.
The main objective of this system is to manage customer information, track interactions, and improve communication between the organization and customers.

The application helps in organizing customer data, maintaining records efficiently, and supporting better decision-making.

## Features :

Customer data management

Add / update / delete customer records

Track customer interactions

User-friendly interface

Secure data handling
## Technologies Used :

Frontend: HTML, CSS, JavaScript

Backend: ( PHP )

Database: MySQL 

Tools: VS Code

## How to Run the Project :

Download or clone the repository.

Move the project folder to htdocs (XAMPP).

Import the database into phpMyAdmin.

Configure database connection in include folder.

Start Apache and MySQL.

Open browser and run: http://localhost/crmprojects/
